    Name: embed object

    Browser View: (Tip - Functionality depends on browser support for .mid files)

    <noembed><bgsound src="yourfile.mid" loop="1"></noembed>

    Browser View: Music will begin playing when your page is loaded and will only play one time. A control panel will be displayed to enable your visitors to stop the music.

    Metadata Tags (`<meta>`)

    Name: meta

    These tags provide metadata about the HTML document and are not directly displayed on the page.

    Meta Description

    <meta name="Description" content="Description of your site">

    Browser View: Nothing will show (Tip)

    Meta Keywords

    <meta name="keywords" content="keywords describing your site">

    Browser View: Nothing will show (Tip)

    Meta Refresh

    <meta HTTP-EQUIV="Refresh" CONTENT="4; URL=http//www.yourdomain.com/">

    Browser View: Nothing will show (Tip)

    Meta Pragma (No Cache)

    <meta http-equiv="Pragma" content="no-cache">

    Browser View: Nothing will show (Tip)

    Meta Rating

    <meta name="rating" content="General">

    Browser View: Nothing will show (Tip)

    Meta Robots (Index/Follow)

    <meta name="robots" content="all">

    Browser View: Nothing will show (Tip)

    Meta Robots (No Index, Follow)

    <meta name="robots" content="noindex, follow">

    Browser View: Nothing will show (Tip)
